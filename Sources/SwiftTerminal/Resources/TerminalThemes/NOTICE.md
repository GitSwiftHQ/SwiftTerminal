Built-in themes in `iTerm2Themes.json` are generated from the upstream
`mbadolato/iTerm2-Color-Schemes` repository:

- Repository: https://github.com/mbadolato/iTerm2-Color-Schemes
- Upstream license for the repository collection: MIT
- Upstream license note: the copyright and license for each individual theme
  belongs to that theme's author

When refreshing the catalog, regenerate it with:

```bash
python3 Scripts/generate_iterm2_theme_catalog.py /path/to/iTerm2-Color-Schemes
```
